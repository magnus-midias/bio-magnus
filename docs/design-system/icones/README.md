# Icones

Iconografia customizada do projeto.

## Formatos aceitos
- SVG (preferencial e padrao).
- PNG transparente so em casos onde SVG nao e possivel.

## Nomenclatura (ASCII puro, kebab-case, sem acentos)

```
icone-<nome-do-icone>[-<variacao>].svg
```

Exemplos:
- `icone-menu.svg`
- `icone-seta-direita.svg`
- `icone-usuario-solido.svg`
- `icone-usuario-contorno.svg`

## Dicas
- Manter viewBox consistente (ex.: `0 0 24 24`) para facilitar reuso.
- Evitar cores fixas no SVG quando possivel; usar `currentColor`.
